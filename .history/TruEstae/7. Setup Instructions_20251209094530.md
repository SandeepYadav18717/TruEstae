### **Endpoint 2: Filter & Sort Data**

```
URL: POST http://localhost:5000/get-data
```

**Request Body:**
```json
{
  "region": "North",
  "gender": "Male",
  "age": "26-35",
  "category": "Clothing",
  "tags": "fashion",
  "payment": "Credit Card",
  "date": "2024-01-15",
  "sortBy": "name-asc",
  "page": 1,
  "limit": 10
}
```

**Parameters:**
- `region` - Filter by region
- `gender` - Filter by gender
- `age` - Filter by age range (e.g., "18-25")
- `category` - Filter by product category
- `tags` - Filter by tags
- `payment` - Filter by payment method
- `date` - Filter by specific date (format: YYYY-MM-DD)
- `sortBy` - Sort option (see below)
- `page` - Page number (default=1)
- `limit` - Results per page (default=10)

**Sort Options:**
- `name-asc` - Sort by customer name A to Z
- `name-desc` - Sort by customer name Z to A
- `newest` - Sort by date (newest first)
- `oldest` - Sort by date (oldest first)

**Response:**
```json
[
  {
    "Customer Name": "Alice Smith",
    "Phone Number": "9123456789",
    "Gender": "Female",
    "Age": 32,
    "Customer Region": "North",
    "Product Category": "Clothing",
    "Tags": "fashion",
    "Payment Method": "Credit Card",
    "Date": "2024-01-15"
  }
]
```

##  COMMON ISSUES & SOLUTIONS

### **Issue 1: "Python not found"**
```
Error: 'python' is not recognized
```
**Solution:**
- Python not in PATH
- Reinstall Python and check "Add Python to PATH"
- Use `python3` instead of `python`

### **Issue 2: "Module not found: pandas/fastapi"**
```
Error: No module named 'pandas'
```
**Solution:**
```bash
pip install --upgrade pip
pip install pandas fastapi uvicorn
```

### **Issue 3: Port 5000 already in use**
```
Error: Address already in use
```
**Solution:**
```bash
# Use different port
python -m uvicorn sort_excel:app --port 5001 --reload
# Then update frontend to use 5001
```

### **Issue 4: Port 5173 already in use**
```bash
npm run dev -- --port 5174
```

### **Issue 5: CSV file not found**
```
Error: FileNotFoundError: 'truestate_assignment_dataset.csv'
```
**Solution:**
- Ensure CSV file is in Backend folder
- Check file name is spelled correctly
- File should be in same folder as sort_excel.py

### **Issue 6: npm modules error**
```bash
# Clear cache
npm cache clean --force

# Delete node_modules
rmdir /s /q node_modules

# Reinstall
npm install
```

### **Issue 7: Cannot connect to backend from frontend**
- Ensure backend is running on port 5000
- Check Home.jsx has correct URL: `http://localhost:5000`
- Check CORS is enabled in sort_excel.py  (it is)
